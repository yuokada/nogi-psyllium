# サイリウムカラー表示 Web アプリケーション 仕様書 v1.0

------------------------------------------------------------------------

# 1. 目的

CSVで管理されたアイドルのサイリウムカラー情報をWeb上で一覧・検索・期別フィルタ表示する静的サイトを構築し、GitHub上で自動デプロイする。

------------------------------------------------------------------------

# 2. システム構成

## 2.1 アーキテクチャ

-   フロントエンド：SPA（静的サイト）
-   データソース：リポジトリ内CSV
-   ホスティング：GitHub Pages
-   CI/CD：GitHub Actions

```{=html}
<!-- -->
```
    GitHub Repository
     ├ data/members.csv
     ├ src (React + TypeScript)
     └ GitHub Actions → build → GitHub Pages公開

# 3. 使用ソフトウェア

## 3.1 開発環境

-   Node.js 20系
-   npm 10系
-   TypeScript 5.x
-   Vite 5.x
-   React 18.x
-   PapaParse（CSV解析ライブラリ）

## 3.2 CI/CD

-   GitHub Actions
    -   actions/checkout@v4
    -   actions/setup-node@v4
    -   actions/upload-pages-artifact@v3
    -   actions/deploy-pages@v4

## 3.3 実行環境

-   GitHub Pages（静的ホスティング）
-   対応ブラウザ：Chrome / Edge / Safari 最新版

------------------------------------------------------------------------

# 4. 機能要件

## 4.1 基本表示機能

-   CSVを読み込み一覧表示
-   各メンバー表示項目
    -   名前
    -   期（任意）
    -   サイリウムカラー1（色名＋背景色）
    -   サイリウムカラー2（任意）
    -   アイコン画像（任意）

## 4.2 検索機能

-   名前・期・色名による部分一致検索

## 4.3 フィルタ機能

-   期別ドロップダウンフィルタ

## 4.4 レイアウト仕様

-   カード形式（レスポンシブ対応）
-   色はHEXコードで背景反映
-   背景色に応じて文字色を自動調整（輝度計算）

------------------------------------------------------------------------

# 5. データ仕様

## 5.1 CSV形式（UTF-8）

    name,gen,color1_name,color1_hex,color2_name,color2_hex,icon

### 必須項目

-   name
-   color1_name
-   color1_hex

### 任意項目

-   gen
-   color2_name
-   color2_hex
-   icon（public配下のパス）

## 5.2 色仕様

-   HEX形式（#RRGGBB）
-   フロントエンド側で簡易妥当性チェックを実施

------------------------------------------------------------------------

# 6. 非機能要件

## 6.1 パフォーマンス
